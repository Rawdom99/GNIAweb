# GNIA - Gastronomic Neural IA 🚀

**GNIA** es una plataforma avanzada diseñada para transformar la gestión operativa y financiera de la industria gastronómica a través de la Inteligencia Artificial. Desde chefs independientes hasta grandes cadenas de restaurantes, GNIA ofrece herramientas precisas para el costeo de recetas, control de inventarios y optimización de ventas.

## 🌐 Demo En Vivo
Puedes ver la plataforma en funcionamiento aquí: [https://Rawdom99.github.io/GNIAweb/](https://Rawdom99.github.io/GNIAweb/)

## ✨ Características Principales
- **Gestión de Recetas:** Costeo automatizado y estandarización de procesos.
- **Asistente IA:** Integración de redes neuronales para análisis predictivo en cocina.
- **Planes Escalables:** Soluciones desde el plan FREE hasta ENTERPRISE para múltiples sedes.
- **Interfaz Premium:** Diseño moderno, responsivo y optimizado para una experiencia de usuario fluida.

## 🛠️ Tecnologías Utilizadas
- **Core:** HTML5, CSS3, JavaScript (ES6+).
- **Assets:** Google Fonts (Outfit), Bootstrap Icons.
- **Control de Versiones:** Git & GitHub.
- **Hosting:** GitHub Pages.

## 📁 Estructura del Proyecto
- `index.html`: Página principal y perfiles de usuario.
- `features-*.html`: Detalle de funciones para cada plan (Free, Flash, Pro, Gold, Elite, Enterprise).
- `pricing.html`: Panel de precios dinámico con selector de moneda.
- `src/`: Carpeta de recursos (Estilos, Scripts, Imágenes).
