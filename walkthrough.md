# GNIA Landing Page - Design Checkpoint

## Current Status (2026-02-01)
The landing page has been significantly refined with a focus on "Software 360°" positioning, premium aesthetics (Metal & Neon), and a specialized navigation system for different business profiles.

### Key Visual & Functional Upgrades:

1. **Hero Section Refinement**:
   - Subtitle color changed to white.
   - Font sizes reduced for a more sophisticated look.
   - Sharp 4:5 aspect ratio for device mockups.

2. **Premium Highlight Section**:
   - Located immediately after the Hero.
   - Text: "Para Chefs Independientes y restaurantes con equipos de 1 hasta 20 personas."
   - **Design**: Golden transparent background (Glassmorphism) with black text for high contrast.

3. **Optimized Tabbed Navigation**:
   - **3 Core Categories**: Chef Independiente, Restaurantes Pequeños, Restaurantes Medianos y Grandes.

   - **Mobile-First UX**: Implemented a custom **Dropdown Selector** for navigating categories on mobile.

4. **Integrated CTAs**:
   - Secondary CTA block added below the tabs section.
   - Maintains Hero icons and styles.
   - Responsive layout: Horizontal (Desktop) / Vertical (Mobile).

5. **Linguistic Polish**:
   - Comprehensive grammar check applied to the entire site.
   - Corrected accents, pluralization, and technical terms (WhatsApp, Excel).
